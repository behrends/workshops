# React Native Teil 1

Auch mit React Native werden wir die Todo-App entwickeln. Zunächst 
lernen wir die Tools von Expo und den prinzipiellen Umgang mit 
plattformübergreifenden Projekten in React Native kennen.

In React Native haben wir es mit UI-Komponenten zu tun. Wir schreiben
eigene Komponenten nutzen und vorgegebene Komponenten wie z.B.
`FlatList`.

Anschließend lernen wir den Umgang mit Zustand (state) in React Native kennen.
