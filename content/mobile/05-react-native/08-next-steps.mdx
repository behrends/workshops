## Datenspeicherung

Im Videokurs gibt es Inhalte zur Datenspeicherung mit

- [AsyncStorage](https://docs.expo.dev/versions/latest/sdk/async-storage/) 
- [SQLite](https://docs.expo.dev/versions/latest/sdk/sqlite/)
- [Firebase](https://docs.expo.dev/guides/using-firebase/)

Eine Alternative zu `AsyncStorage` ist der _Key-Value-Storage_ 
mit [Expos SQLite-API](https://docs.expo.dev/versions/latest/sdk/sqlite/#key-value-storage)

## Navigation mit expo-router

[Expo Router](https://docs.expo.dev/router/introduction/) ermöglicht dateibasierte Navigation in React Native Apps.

Sollten wir Zeit haben, dann könnten wir z.B. eine einfache
Bottom-Navigation mit [Expo Router](https://docs.expo.dev/router/introduction/) umsetzen &mdash; ggf. mit Hilfe von KI-Coding-Tools.

Wenn ein neues Expo-Projekt mit `npx create-expo-app` 
und dem Default- oder Navigation-Template
erstellt wird, dann ist bereits eine Navigation mit
expo-router eingerichtet.

<Callout type="warning">
Im Videokurs wird die Navigation mit [React Navigation](https://reactnavigation.org) umgesetzt. Expo Router ist eine neuere Alternative, die auf React Navigation aufbaut und eine dateibasierte Navigation ermöglicht.
</Callout>
