# Progressive Web Apps

In dieser Vorlesung berücksichtigen wir Progressive Web Apps (PWAs) nicht, 
weil dieser Ansatz im Wesentlichen der Entwicklung von Webanwendungen entspricht, 
was in vorigen Vorlesungen bereits behandelt wurde.

Für interessierte hier einige Links zu weiterführenden Informationen:

- [Infos über PWAs bei web.dev (Google)](https://web.dev/progressive-web-apps/)
- [Kurs zu PWAs bei web.dev (Google)](https://web.dev/learn/pwa/)
- [Infos über PWAs im Mozilla Developer Network](https://developer.mozilla.org/en-US/docs/Web/Progressive_web_apps)
- [PWA builder von Microsoft](https://www.pwabuilder.com)
- [„Get started with PWA“ von Mircosoft](https://learn.microsoft.com/en-us/microsoft-edge/progressive-web-apps-chromium/how-to/)
- [Capabilities project &mdash; making it possible for web apps to do anything iOS, Android, or desktop apps can („Project Fugu“ von Google)](https://developer.chrome.com/docs/capabilities)
- Web Components mit [Lit](https://lit.dev) oder [Stencil](https://stenciljs.com)
