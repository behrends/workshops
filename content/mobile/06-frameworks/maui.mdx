# .NET Maui

## Allgemeines

https://dotnet.microsoft.com/en-us/apps/maui

- Obwohl Microsoft React Native unterstützt hat das Unternehmen mit .NET Maui ein eigenständiges Framework.
- Maui steht für _Multi-platform App UI_.
- Apps werden in C# geschrieben

## Installation 

Siehe https://dotnet.microsoft.com/en-us/learn/maui/first-app-tutorial/intro

Auf Windows wird Visual Studio 2022 mit der .NET MAUI workload benötigt. 

In macOS kann VS Code mit Erweiterungen genutzt werden, wobei zuvor das .NET SDK installiert werden muss.

Stand 2025 führen wir dies nicht vor, weil die Installation zu umständlich erscheint. 

**Dennoch könnte dies ein gut geeignetes Framework für diejenigen sein, die gerne mit C# arbeiten.**
