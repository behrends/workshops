## Firebase

Vorbemerkung: Firebase kann kostenlos genutzt werden. Leider ist jedoch
für Firebase Cloud Storage eine Kreditkarte zur Verifikation der Identität
notwendig, auch wenn nur die kostenlose Stufe genutzt wird. Firebase Cloud
Storage wird jedoch benötigt, um Dateien wie z. B. Bilder in Firebasehochzuladen.

Zu Firebase gibt es ein Beispiel-Projekt: https://github.com/behrends/firebase-expo. 
Das Readme im Projekt beschreibt die Vorgehensweise zur Einrichtung des Firebase-Projekts 
und zur Nutzung der Firebase-Dienste im Code.
