# Android und KMP

Die Weiterentwicklung der Beispiel-App (Todos) für Android wird in 
diesem Teil fortgesetzt:

- Datenbanken mit Room verwalten und verwenden
- Navigationsstruktur vorbereiten
- Links zu Lernressourcen
- Vorstellung von KMP (Kotlin Multiplatform Mobile)

Je nach verfügbarer Zeit gibt es noch einen Ausblick auf 
weitere Themen.
