# Prettier (Formatierung)

<Callout>
  **Dauer:** 15 Minuten

  - Prettier formatiert Code automatisch
  - Einheitliche Formatierung im gesamten Projekt

  **Ziel:** Prettier installieren und konfigurieren
</Callout>

## Was ist Prettier?

[Prettier](https://prettier.io) ist ein Code-Formatter, der
JavaScript, TypeScript, HTML, CSS, JSON und viele weitere
Sprachen automatisch formatiert. Prettier sorgt dafür, dass
der gesamte Code in einem Projekt einheitlich aussieht &mdash;
unabhängig davon, wer ihn geschrieben hat.

## Installation

Prettier wird als Entwicklungsabhängigkeit installiert:

```bash
npm install --save-dev prettier
```

## Konfiguration

Prettier kann über eine Datei `.prettierrc` im Projektordner
konfiguriert werden:

```json
{
  "singleQuote": true,
  "semi": true,
  "trailingComma": "all"
}
```

Wichtige Optionen:

- `singleQuote`: Einfache statt doppelte Anführungszeichen
- `semi`: Semikolons am Zeilenende
- `trailingComma`: Komma nach dem letzten Element in Listen

Alle Optionen sind in der [Dokumentation](https://prettier.io/docs/options)
beschrieben.

## Vorher/Nachher

Vorher (unformatiert):

```js
function greet( name){
console.log("Hallo, "+name+"!")
    const x=42
  if(x>10){console.log(  "groß")}
}
```

Nachher (mit Prettier formatiert):

```js
function greet(name) {
  console.log('Hallo, ' + name + '!');
  const x = 42;
  if (x > 10) {
    console.log('groß');
  }
}
```

## VS Code Extension

Es gibt eine offizielle [Prettier-Extension für VS Code](https://marketplace.visualstudio.com/items?itemName=esbenp.prettier-vscode).

Nach der Installation kann Prettier als Standard-Formatter
eingestellt werden. In den VS Code Settings kann die Option
**Format on Save** aktiviert werden, sodass Dateien beim
Speichern automatisch formatiert werden.

<Callout type="info">
In der Datei `.vscode/settings.json` im Projektordner kann
die Formatierung beim Speichern projektspezifisch aktiviert werden:

```json
{
  "editor.defaultFormatter": "esbenp.prettier-vscode",
  "editor.formatOnSave": true
}
```
</Callout>

## npm Script

Zum Formatieren aller Dateien im Projekt kann ein Script
in der `package.json` definiert werden:

```json
{
  "scripts": {
    "format": "prettier --write ."
  }
}
```

Ausführung mit:

```bash
npm run format
```
