# Vorführung der Webanwendungen

Dies ist der letzte Termin, der idealerweise in der letzten Woche vor der Klausurwoche stattfindet, damit alle Gruppen möglichst weit in der Entwicklung ihrer Projekte sind.

Jede Gruppe stellt in 15-20 Minuten den aktuellen Stand
ihrer Webanwendung vor &mdash; auch der Code soll gezeigt werden, um eine gemeinsame Diskussion im Kurs anzustoßen.

Danach können letzte Fragen vor der Abgabe der Projekte
gestellt werden.

Die restliche Zeit wird für das Programmieren genutzt.
