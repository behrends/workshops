# CSS Teil 1

Hier wird eine Einführung in CSS mit folgenden Teilthemen gegeben:

- CSS-Syntax: Regeln, Selektoren, Deklarationen
- Einbindung von CSS in HTML: 
  - Inline dem `style`-Attribut, 
  - Internal mit dem `<style>`-Tag, 
  - External mit dem `<link>`-Tag
- Wichtige CSS Selektoren: Element, ID, Class
- Beispiele für CSS Deklarationen: Farbe, Schrift, Hintergrund, …
