# Live Coding (Wetter-App)

<Callout>
  **Dauer:** 30 Minuten

  **Ziel:** Nächste Schritte in der gemeinsamen Entwicklung einer Beispiel-App im Live Coding zum Mitmachen.
</Callout>

Die Beispiel-App (Wetter) wird gemeinsam in replit.com weiterentwickelt:

- Funktionen einbauen (u.a. `input`, `output`)
- `if` für den Test auf gültige Zahleingabe (mit `parseInt()` und `isNaN()`)
- evtl. Ausblick auf `while`-Schleife für die wiederholte Eingabe bei ungültiger Zahl
