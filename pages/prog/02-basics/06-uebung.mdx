# Übungen

<Callout>
  **Dauer:** 60 Minuten bzw. restliche Zeit \
  40 Minuten inklusive Pause \
  20 Minuten Besprechung \
  (evtl. Besprechung beim nächsten Termin)

  **Ziel:** Variablen, Berechnungen, Ein- und Ausgabe
</Callout>

<Callout type="warning">
Auch in Moodle stellen (evtl. inklusive Abgabe).
</Callout>

Übungen werden in Moodle bereitgestellt.

<Callout type="info">
Dozentenhinweis: Aufgaben und Lösungen siehe Repository `exercises`.
</Callout>

<Callout>
Idealerweise sollten alle inzwischen Node.js und VS Code installiert
haben. Falls nicht, dann kann die Übung auch in 
[replit.com](https://replit.com) durchgeführt werden.

Bei Bedarf kann Hilfe bei der Installation und Einrichtung 
gegeben werden.
</Callout>
