# Programme strukturieren

<Callout>  
  **Ziel:** Grundstruktur des Programms festlegen
</Callout>

Mit Dateien und Funktionen kann eine 
Grundstruktur des Programms festgelegt werden.

Für jeden „Bereich“ (Verwaltungsmodus, 
Anwendungsmodus, usw.) kann dabei z.B.
eine Datei mit einer oder mehreren Funktionen
erstellt werden. Jede konkrete „Aufgabe“ im 
Programm (z.B. Hauptmenü, Daten laden, usw.)
kann mit einer bestimmten Funktion umgesetzt
werden.
