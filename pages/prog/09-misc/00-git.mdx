# Git

Git ist ein verteiltes Versionskontrollsystem, das von Linus Torvalds für die Entwicklung des Linux-Kernels entwickelt wurde. Es ist ein Open-Source-System, das es ermöglicht, Änderungen an Dateien zu verfolgen und zu speichern. Es ist ein sehr mächtiges Werkzeug, das von Entwicklern auf der ganzen Welt verwendet wird, um ihre Projekte zu verwalten.
