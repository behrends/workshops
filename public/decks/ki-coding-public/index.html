Vibe Coding vs. Agentic Engineering

Vibe Coding

Schnell, explorativ, oft spielerisch

"I just see stuff, say stuff, run stuff, and copy paste stuff."

Karpathy, Februar 2025. Gut für Prototypen, kleine Apps und persönliches Ausprobieren.

Agentic Engineering

Planvoll, reviewbar, für wichtige Software

Auch als augmented coding oder AI assisted engineering beschreibbar.

Geeignet, wenn andere Menschen von der Qualität und Stabilität abhängig sind.

Zwei sinnvolle Modi: Apps für mich selbst anders bauen als Software, die für andere zuverlässig funktionieren muss.
Zwischenfrage für den Vortrag: Wollen wir an dieser Stelle etwas vorführen oder ist das Material bereits dicht genug?

Konsequenzen

1

Softwareentwicklung ist mehr als Coding

Anforderungen, Architektur, Review, Sicherheit und Betrieb bleiben entscheidend.

2

Coding Agents schreiben schneller und oft besser

Vor allem bei Standardmustern, Migrationen und wiederkehrenden Aufgaben.

3

Code wird billiger

Mehr Zeit kann in die wirklich teuren Probleme fließen: Qualität, Produkt und Verantwortung.

4

Hohe Qualität bleibt anspruchsvoll

Gute Systeme entstehen nicht automatisch, nur weil mehr Code produziert wird.

KI codet, wer überprüft?

Wenn Output explodiert, wird Review selbst zum Engpass.

Human in the Loop

Menschliche Kontrolle bleibt wichtig, skaliert aber schlecht, wenn sehr viel neuer Code entsteht.

Menge

Zu viel neuer Code

Vollständige manuelle Prüfung wird schnell langsam und teuer.

Wirkung

Kaum Produktivitätsgewinn

Wenn alles erneut von Hand geprüft werden muss, verschiebt sich der Flaschenhals nur.

Review wird selbst agentisch. Das kann besser skalieren als ein Mensch allein im Loop.

Ausblick

Arbeitsmarkt

Weniger manuelles Tippen, nicht weniger Arbeit

Jevons-Paradox, mehr Softwarebedarf und neue Rollen sprechen eher für Verschiebung als für Stillstand.

Breite Nutzung

Skill Atrophy

Verlernen verhindern

Systeme wie Code Companion können helfen, aktives Lernen und bewusstes Verstehen zu erhalten.

Modelle

Neben Codex und Claude spielen Gemini, Gemma, Modelle aus China und Mistral aus Europa eine Rolle.

Agentic Coding könnte eine weitere Ebene der Abstraktion sein. Die Verantwortung verschwindet dadurch nicht.
