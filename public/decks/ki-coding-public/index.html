Was ändert sich?

KI-Agents können viele Standardaufgaben schneller erledigen und erzeugen bei guter Steuerung oft brauchbaren Code. → Ende der manuellen Programmierung?

Code ist seltener der Engpass und kann viel günstiger und schneller produziert werden.

Die Schnittstelle verschiebt sich weg vom direkten Schreiben in Programmiersprachen.

Agentic Engineering bewirkt eine weitere Abstraktion der Programmiersprachen.

Was bleibt unsere Aufgabe?

Weitere Themen

Jevons-Paradox

Wenn eine Ressource effizienter nutzbar wird, sinken die Kosten pro Einheit und die Gesamtnutzung kann sogar steigen.

Übertragen auf KI-Coding

Code wird günstiger

Mehr Menschen und Teams können mehr Software bauen als zuvor.

Folge

Mehr Produkte, mehr Varianten, mehr Integration

Das kann den Bedarf an Engineering insgesamt erhöhen statt nur zu reduzieren.

Vibe Coding vs. Agentic Engineering

Vibe Coding

"I just see stuff, say stuff, run stuff, and copy paste stuff." — A. Karpathy (ex OpenAI, Tesla), Feb 2025

Sinnvoll für Prototypen, „Wegwerf“-Apps und persönliches Ausprobieren.

Agentic Engineering

„Echtes“ Software Engineering für produktive Apps. Der Mensch steuert, überprüft und gestaltet die Software mit KI-Assistenz. Programmcode wird von Agents generiert und getestet.

Ist die Software nur für mich oder muss sie für andere zuverlässig funktionieren?
