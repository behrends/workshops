Menschliche Schwachstelle

Vollständige manuelle Prüfung ist langsam, unvollständig und unzuverlässig.

Wirkung

Der Flaschenhals verschiebt sich und die gewonnene Produktivität sinkt wieder.

Menschliche Kontrolle und Entscheidungen bleiben wichtig, skalieren aber schlecht, wenn sehr viel neuer Code entsteht.
Ähnlich wie beim autonomen Fahren machen Agents weniger Fehler.

Problem: Agents bleiben nicht-deterministisch

Sandboxes

Agents sollen gar nicht auf sensitive Daten oder Produktionssysteme zugreifen.
Wir sollten uns bewusst sein, dass KI-Agents nur im Yolo-Mode (Autopilot o. a.) den vollen Nutzen entfalten können, aber auch die größten Risiken bergen.
Eigene Maschinen, VMs oder Container.
Agents dürfen nur den Code und die Dateien anpassen, die wir ihnen geben. Aber damit dürfen sie frei arbeiten und experimentieren mit vollen Systemrechten, ohne dass wir ständig eingreifen müssen.

Abschluss

KI-Agents können fast jeden Code besser und schneller generieren als Menschen.

Code ist kein Engpass mehr und kann deutlich günstiger und schneller produziert werden.

Agentic Coding ist eine weitere Ebene der Abstraktion der Programmiersprachen.

Softwareentwicklung umfasst viel mehr als nur die Programmierung.

Die Verantwortung für die Software bleibt bei den Menschen.
